## 需要的环境

**浏览器**（推荐使用谷歌浏览器）

## 功能介绍

原理：使用 Ajax 发送 Post/Get 请求模拟正常发帖等操作
现有功能：自动 访问公共群主页、发布微社区、投票

## 使用说明

脚本：JavaScript 文件夹下的 ajax.js 文件
用法：在电脑登陆易班网页版后，打开浏览器控制台，复制 **ajax.js** 文件内的内容到控制台回车执行！

## 接口调用

使用 getJSON 方法

    $.getJSON("接口地址",

    });
